# 🚀 My Personal Website

这是一个追求极致视觉表现力与性能的个人门户网站。基于 **Next.js** 生态系统构建，融合了复杂的物理动画与现代化的响应式设计。

---

## 🌟 核心特性

- **流畅交互：** 结合 `GSAP` 的时间轴控制与 `Framer Motion` 的手势交互。
- **现代美学：** 使用 `Tailwind CSS` 实现的响应式布局，完美适配各种屏幕尺寸。
- **性能优化：** 利用 Next.js 的服务端渲染 (SSR) 和图像优化功能。
- **类型安全：** 全量使用 TypeScript 编写，确保代码的健壮性。

---

## 🛠️ 技术栈

| 领域         | 技术方案                                            |
| :----------- | :-------------------------------------------------- |
| **基础框架** | [Next.js](https://nextjs.org/) (App Router)         |
| **视图库**   | [React](https://reactjs.org/)                       |
| **样式处理** | [Tailwind CSS](https://tailwindcss.com/)            |
| **物理动画** | [Framer Motion](https://www.framer.com/motion/)     |
| **滚动效果** | [GSAP](https://greensock.com/gsap/) (ScrollTrigger) |
| **包管理**   | [pnpm](https://pnpm.io/)                            |
